Face Recognition Training

This program pics a random image from a set list and then searches through the image for faces. It cuts out each face and places an input box next to it with the placeholder set to whoever it thinks is in the image. If you leave the input box blank, nothing will be changed. If you type a name into the input box, the face will be added to the database under that name.

As the program is looking through the photos for faces on each page load, it can take as much as 30 seconds for the page to load.
